### With Icon

Using an icon replaces a text or combination.

<!--start-code-->

```jsx
render(
  <Breadcrumb>
    <Breadcrumb.Item>
      <Icon type="Home" />
    </Breadcrumb.Item>
    <Breadcrumb.Item>
      <a href=""><Icon type="Inspection" />Project List</a>
    </Breadcrumb.Item>
    <Breadcrumb.Item>A Project</Breadcrumb.Item>
  </Breadcrumb>
);
```

<!--end-code-->

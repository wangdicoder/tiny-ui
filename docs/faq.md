# FAQ

Here are the frequently asked questions about Tiny UI.

### What are features of this project?

- This project is implemented by TypeScript
- All components are implemented by React function components using [React Hooks](https://react.dev/reference/react/hooks)
- All form components support controlled and uncontrolled usages. [What's Uncontrolled Components?](https://react.dev/learn/sharing-state-between-components#controlled-and-uncontrolled-components)

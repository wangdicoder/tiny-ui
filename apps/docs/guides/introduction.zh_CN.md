# Tiny UI

Tiny UI 是一套基于 React.js **18+** 构建的 UI 组件和控件库，主要服务于中后台产品。

## 特性

- 高质量的 React 组件。
- 支持自定义主题。
- 使用 TypeScript 编写，提供完整的类型定义。
- 完全采用 React 函数组件，基于 [React Hook](https://react.dev/reference/react/hooks) 实现。
- 严格遵循 [WAI-ARIA](https://www.w3.org/WAI/standards-guidelines/aria/) 无障碍标准。

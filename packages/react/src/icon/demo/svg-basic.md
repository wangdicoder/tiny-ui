### Basic

Import individual SVG icon components. Each icon is tree-shakeable.

```jsx live
() => {
  return (
    <div style={{ display: 'flex', gap: 16, alignItems: 'center', fontSize: 24 }}>
      <IconClose />
      <IconPlus />
      <IconSearch />
      <IconHeart />
      <IconStar />
    </div>
  );
}
```

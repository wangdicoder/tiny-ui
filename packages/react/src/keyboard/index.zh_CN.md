# Keyboard

键盘样式按钮。

## 使用场景

- 展示键盘操作。

## 引入方式

```jsx
import { Keyboard } from 'tiny-design';
```

## API

| 属性      | 说明                 | 类型          | 默认值  |
| --------- | -------------------- | ------------- | ------- |
| children  | 键盘按键标签         | ReactNode     | -       |
| style     | 容器的样式对象       | CSSProperties | -       |
| className | 容器的类名           | string        | -       |
